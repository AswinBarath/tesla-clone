## What's unique in this Project

- The Tesla clone project is an attempt to replicate the Tesla cars landing page with React JS and React ecosystem.
- Tesla clone uses redux for managing states such as user clicks for checking hamburger menu, passing tesla cars data.
- Styled components are used for developer productivity where components are wrapped in containers and styles are applied inside the same component codebase.
- [Material UI](https://mui.com/) and [React Reveal](https://www.react-reveal.com/) libraries are used to create decent animations for easy user experience in scrolling the landing page.
- The clone is deployed on Heroku cloud platform with continous build processes once GitHub repo code is updated.
